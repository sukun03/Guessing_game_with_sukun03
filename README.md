# 🎯 Number Guessing Game (Python)

A simple Python-based number guessing game where the player must guess a randomly generated number within **20 seconds**.

The game provides hints after each attempt and ends automatically when the time runs out.

---

## 📌 Features

- Random number generated between 1 and 100
- Multiple attempts allowed within time limit
- 20-second countdown timer
- Hints after each guess (higher or lower)
- Game ends automatically when time finishes
- Beginner-friendly Python project

---

## 🛠️ Technologies Used

- Python
- Random Module
- Time Module

---

## ▶️ How to Run the Game

### Step 1: Clone the repository

```bash
git clone https://github.com/sukun03/Guessing_game_with_sukun03.git
```

### Step 2: Navigate to project folder

```bash
cd Guessing_game_with_sukun03
```

### Step 3: Run the program

```bash
python guessing_game.py
```

---

## 🎮 How to Play

1. Start the game
2. Enter **Y** to play
3. Guess a number between **1 and 100**
4. Follow hints provided by the program
5. Guess correctly before **20 seconds** ends
